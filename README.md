# Menstrual Health Chatbot

A comprehensive menstrual health chatbot application with React frontend and Flask backend, featuring RAG (Retrieval-Augmented Generation) capabilities for accurate health information delivery.

## Project Structure

```
is_project2_final/
├── is_project2/
│   ├── backend/          # Flask backend with RAG pipeline
│   ├── src/              # React frontend application
│   └── public/           # Static assets
```

## Features

- **Frontend**: React + Vite application with Firebase authentication
- **Backend**: Flask API with RAG pipeline using fine-tuned Flan-T5 model
- **Multi-language Support**: English and Swahili
- **Vector Search**: FAISS-based semantic search for accurate information retrieval

## Development Workflow

This project uses a branch-based development workflow:
- `main`: Production-ready code
- `frontend`: Frontend development branch
- `backend`: Backend development branch

See individual README files in frontend and backend directories for setup instructions.
